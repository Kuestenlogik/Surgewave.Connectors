# Code of Conduct

Surgewave adopts the **Contributor Covenant, version 2.1** as its code of conduct. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

Concerns about behaviour in the project's spaces (issues, pull requests, discussions, Discord, email threads) can be sent to the maintainers at:

**conduct@kuestenlogik.de**

Reports are handled confidentially. Maintainers will respond within five business days and will follow the enforcement guidelines published at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines>

## Scope

This Code of Conduct applies to all project spaces — repositories under the `Kuestenlogik` organisation, the public Discord server, the issue tracker, pull-request discussions, and any direct project communication. It also applies when an individual is officially representing the project in public spaces.
